# AWS 배포 사전세팅 (B2팀 백엔드)

## 1. EC2 스펙표
- 인스턴스 타입: t3.small ('프리티어 사용가능' 태그 붙어있다면)
- 리전: 서울 (ap-northeast-2)
- OS: Ubuntu 24.04 LTS
- 스토리지 용량: 30GB

## 2. IAM 사용자 권한 범위 정의
- 우리가 EC2 인스턴스를 생성·운영하는 데 필요한 최소 권한만
- 부여: AWS 관리형 정책 `AmazonEC2FullAccess` 하나만 연결
  (EC2 외 다른 서비스는 접근 불가, root 계정 대신 이 IAM 사용자로 로그인해서 작업)
